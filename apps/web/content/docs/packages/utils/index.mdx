## Overview

This package is the lightweight home for reusable helpers that do not belong to a more specific domain package. The current surface area is intentionally small and can grow as common utilities are extracted from apps and feature packages.

## API

### `getGreeting(name)`

Returns a greeting string for the provided name.

## Usage

```typescript
import { getGreeting } from '@md-oss/utils';

console.log(getGreeting('World'));
// Hello, World!
```
